You can create a hostpath-backed persistent volume with a persistent volume claim like this:

Possible class names supported by this deployment are {{ keys .Values.storageClasses.classes }}

For example:

apiVersion: v1
kind: PersistentVolumeClaim
metadata:
  name: local-path-pvc
spec:
  accessModes:
    - ReadWriteOnce
  storageClassName: {{ (keys .Values.storageClasses.classes | first) }}
  resources:
    requests:
      storage: 2Gi
